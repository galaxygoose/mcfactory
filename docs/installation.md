# Installation

## Prerequisites

- Node.js 18+
- npm or yarn

## Install MCF

```bash
npm install mcf
```

## Configuration

Create a `.env` file with your API keys:

```bash
OPENAI_API_KEY=your_openai_api_key
ANTHROPIC_API_KEY=your_anthropic_api_key
# ... other keys
```

## Verify Installation

```bash
mcf --version
```
